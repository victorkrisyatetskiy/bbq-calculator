# BBQ Calculator

BBQ Calculator is a playful web application that helps you calculate how much meat, snacks, and drinks you need for a BBQ, depending on the number of adults and kids, hunger level, meat types, and party format.

## Why?

This project was created to practice fullstack development skills by building a complete end-to-end application with a modern frontend, REST API backend, business logic, validation, testing, and deployment.

## Tech Stack

Backend:
- Java 20
- Spring Boot
- Maven

Frontend:
- React
- TypeScript
- Vite

Additional tools:
- Docker
- GitHub
- REST API

The stack may evolve during development as new features and requirements are added.

## MVP

Input:
- Number of adults
- Number of children
- Hunger level
- Meat type

Output:
- Required meat amount
- Funny comment

## Features

Current features:
- BBQ meat calculation
- Support for adults and children count
- Different hunger levels
- Meat type selection
- Funny comments based on calculation results

Planned features:
- Funny hunger level names
- Meat, drink, and snack emojis
- Drinks and snacks calculation
- BBQ budget estimation
- Party mode / picnic style presets
